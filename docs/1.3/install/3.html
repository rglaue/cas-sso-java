Find the Source id and name for your institution

About

Blackboard 8 supports many domains within a single installation, and many institutions within a single domain. Each institution controls its own authentication process with a module. So though you install the CAS module for the entire server, you must activate it and configure it on a per-institution basis.

Finding the Source id and name

You need two values related to the institution of interest in order to setup the CAS authentication. There are two ways to obtain the sourceid.id and sourceid.source values. One is through the command line using siapi.sh. The other through the Web Administration interface.

Using siapi.sh:

bash-3.00# ./siapi.sh printInstitutionIds

Your environment has been set.

Source=WebCT, ID=URN:X-WEBCT-VISTA-V1:75d7e9fe-8f2b-de2a-0123-4567899529a7
 glcid=URN:X-WEBCT-VISTA-V1:75d7e9fe-8f2b-de2a-0123-4567899529a7 name=Western Illinois University

bash-3.00#
The printInstitutionIds command returns the glcid of the institution, as well as the sourcedid.id and sourcedid.source of the institution.

Using the Web Administration interface:

sourcedid.source: WebCT
sourcedid.id: URN:X-WEBCT-VISTA-V1:75d7e9fe-8f2b-de2a-0123-4567899529a7
An alternate to finding the sourceid.source: Log on in the Domain Admin role, go to Institutions, click on the ActionLink for the institution you are interested in and choose "View Information"
